Az ember, látván, hogy a LilyPond a klasszikus zene lejegyzésére összpontosít, elbizonytalanodhat afelől, hogy vajon képes-e modern zene lekottázására. Szerencsére a válasz egy határozott igen.

„Nézd, mama, hackelés nélkül ment!”

Bármilyen kotta generálása egyszerű szöveges parancsokkal lehetséges. Ha avantgarde zenét kottázol, biztosan tetszeni fog a Lily rugalmassága az ütemek és az ütemmutatók kezelésében. Íme, itt egy példa, mely egymásba ágyazott n-olákat tartalmaz. A hangok vízszintes elhelyezése pontosan a hosszúságukkal arányos. A hangoknak nem kell pontosan kitölteniük az ütemet, és két hang csak akkor kerül egymás alá, ha pontosan egyszerre szólalnának meg.

Kattints ide a PDF letöltéséhez

Bármely más programban ez jelentős mennyiségű pepecselést igényelne. A LilyPondban egyszerűen csak be kell írni a hangokat a ritmusukkal együtt. 

\times 7/9 {
  \times 4/6 {
     r8 c32[ c c c c c c c] r4
     c32[ c32 c16
  }
  \times 5/4 {
    c16 c c] c32[ c32 c16 c] r8
  }
}

Ha jártas vagy a Scheme programozási nyelvben, akár algoritmikusan komponálhatsz a LilyPondon belül!

“Nézd, mama, igazán könnyű hackek segítségével ment!”

Bármilyen nem szabványos jelenségre lenne szükséged? Apró trükkök segítségével a LilyPond határtalan rugalmasságából szinte bármit ki tudsz hozni, pl. Schenker-diagramokat.
