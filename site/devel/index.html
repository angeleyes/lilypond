I implemented XXXX! How do I get you to include it?

diff -u oldfile newfile and don't send complete files, even if the diff is larger than the whole file.

Don't forget to put your name and e-mail address in the file `Documentation/topdocs/AUTHORS.texi', and add an entry in the ChangeLog.

If you changed more than just one file, making diffs manually can be cumbersome; check out MakingPatches.

How do I inspect `String's and `SCM' values in GDB?

Use the following command definitions, which can be put in the file ~/.gdbinit. define printstr print $arg0.ch_C () end define printscm call gh_display ($arg0) call gh_newline () end
