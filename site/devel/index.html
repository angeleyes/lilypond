LilyPond Development

WARNING: compiling LilyPond is involved. Don't do it if there is a binary for your platform.

Source code

Source code is available
  • By FTP/HTTP from
    • http://lilypond.org/download/
    • http://www.lilypond.org/ftp/
    • ftp://sca.uwaterloo.ca/pub/
  • By CVS from http://savannah.gnu.org/cvs/?group=lilypond.

    The repository does not contain generated files. To create configure, run 

              ./autogen.sh
Instructions for compiling are in the INSTALL instructions.
