LilyPond Development

WARNING: Building LilyPond is an involved process. If possible, download a precompiled binary for your platform.

Source code

Source code is available
  • By HTTP from
    • lilypond.org
    • sca.uwaterloo.ca
  • By CVS from savannah.gnu.org 
          CVS_RSH=ssh cvs -d:ext:anoncvs@savannah.gnu.org:/cvsroot/lilypond co lilypond

    The CVS repository does not contain generated files. To create configure, run 

    	  ./autogen.sh

More information on in the INSTALL documents with building and installation instructions (version 2.6) and building and installation instructions (version 2.7)

Participating

Participating in the LilyPond project is highly encouraged!
