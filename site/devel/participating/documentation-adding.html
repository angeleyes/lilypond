LilyPond Snippet Repository

The LilyPond Snippet Repository (affectionately known as LSR) is a collection of small examples. It can be easily searched for special tricks, or simply browsed to see some neat examples. You can also add your own examples, so that other LilyPond users can benefit from your knowledge. Visit the LSR and see for yourself!

Documentation suggestions

If you have suggestions for the documentation, please be as specific as possible. “The docs for lyrics suck!” isn't a very useful suggestion. Instead, please tell us exactly what to change and where to change it. Do not forward us a discussion from lilypond-user with a note that “this contains info that should be in the manual!” That is not helpful; we need to start reading the whole discussion to figure out what is happening, then read the documentation to see what could be clarified or added. Please take the time to write the exact changes to the text. If you have been active in the discussion, then you probably understand the material better than we do.

A formal patch to the source code is not required. Simply send an email to lilypond-devel@gnu.org with your addition to the documentation. We will take care of the technical details. Here is an imaginary example of a perfect documentation report: 

To: lilypond-devel@gnu.org
From: helpful-user@example.net
Subject: doc addition 

In section 2.13 (printing lyrics), above the last line ("More options, like..."),
please add:
    
----
To add lyrics to a divided part, use blah blah blah.  For example,

\score {
 \notes {blah <<blah>> }
 \lyrics {blah <<blah>> }
 blah blah blah
}
----

In addition, the second sentence of the first paragraph is
confusing.  Please delete that sentence (it begins "Users
often...") and replace it with this:
----
To align lyrics with something, do this thing.
----

Have a nice day,
Helpful User
We may edit your suggestion for spelling, grammar, or style, and we may not place the material exactly where you suggested, but if you give us some material to work with, we can improve the manual much faster.
