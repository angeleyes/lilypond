RELEASE-MEISTER

The job of the release-meister is to build, troubleshoot and upload releases.

LilyPond is ready for a release when:

  • The regression test comes out clean
  • All features work as documented
  • A new release offers an improvement over the last one
The release-meister keeps an eye on the bug tracker and the current head to determine when a release should be made.

The actual mechanics of building a release is completely scripted in our Grand Unified Builder system.
