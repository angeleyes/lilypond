The

CODE JANITOR

makes sure that new C++ and Scheme code follows the standards.

Writing LilyPond code is an intensive --and sometimes frantic-- process, and the code that is written in deep-hack mode often is unpolished. The janitor helps by adding the polish, and standardizing the code

  • Cleanup code spacing, formatting, indenting, file names, etc.
  • Standardize
    • error messages
    • doc strings
    • texidoc strings
    • function comments
    • debugging code
  • Submit pot files to the translation robot, and update the translations in CVS.
  • Define new coding style standards, where necessary.

REQUIREMENTS:

Moderate experience with programming C, Scheme and/or Python
