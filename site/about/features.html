LilyPond's features

This list is incomplete. See the user manual for more information.
  • Free Software
  • Beautiful printed music
  • Runs on GNU/Linux, MacOS X, and all non-prerelease flavours of Microsoft Windows (thanks to Cygwin).
  • PDF output, PostScript output
  • Open, portable input format
  • Active user community
  • Printing text with music by seamless integration with LaTeX
  • GUI support by Denemo, NoteEdit, Rosegarden (experimental)
  • Simple MIDI output
  • Full control through GUILE scripting
  • XML output
  • Scalable music font
  • ABC input filter
