Using LilyPond

Newbies

Start learning by browsing the Tutorial

Experienced users

Check out the documentation, which should be installed in 

  /usr/share/doc/lilypond

If it is not there, browse the documentation on-line. (For other versions than the latest stable release, go here).

Help!

We have a mailing list:

  • For questions, post a message to the lilypond-user@gnu.org mailing list.
  • To subscribe, head over to http://mail.gnu.org/mailman/listinfo/lilypond-user
  • To browse and search, head over to the list archives

Keep in touch

To keep in touch with the latest news, subscribe to our announce mailing list.
