Installing LilyPond

When we have whetted your appetite for LilyPond, then this is the right page. Click the download links for the platform of your choice, and go onto Getting started.

Download LilyPond 2.4 -- stable branch

Note LilyPond 2.4 requires the ec-fonts-mftraced package from here @stable.ihtml@
Platform Version Go here LilyPond version Packager

Download LilyPond 2.5 -- development branch

Note LilyPond 2.5 requires version 8.15 or newer of Ghostscript. @devel.ihtml@
Platform Version Go here LilyPond version Packager
