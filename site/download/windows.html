LilyPond on MS Windows

Installation

  • Click Install Lily now,
  • For NT, 2000 and XP: become administrator
  • Run the installer (setup.exe)
  • Keep clicking "Next" until
  • in the Package View dialogue, select LilyPond, found in the Publishing section.

If you are not 100% exactly sure of what you're doing, then just click Next, don't change the default install directory (c:/cygwin). don't change the default text type (UNIX).

Testing the install

If you are using version 1.8.2-1 or newer, try this first. Download this first .ly example:

example-1.ly
to your desktop. It should be called: 'example-1.ly' (not example-1.ly.TXT). Double-click on the .ly file on your desktop. LilyPond should generate the file and open your favourite PDF viewer.

See getting started for more information and how to create your first LilyPond .ly music file.

Uninstalling

  • By brute force: remove C:\CYGWIN. This also removes your Cygwin /home/ directory, so copy important files to somewhere else first.,
  • Officially:

    Start setup.exe, click Next, select Install from Local Directory. Then click next until you reach Select packages to install. Click View. Now, for every package, click on the recycle icon until it says Uninstall. Then, click Next.
