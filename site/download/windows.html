LilyPond on MS Windows

1. Installation

  • Download setup.exe by shift-clicking here,
  • For NT, 2000 and XP: become administrator
  • Run setup.exe.
  • Keep clicking "Next" until
  • in the Package View dialogue, select LilyPond, found in the Publishing section.
  • LilyPond will now be installed, along with supporting programs. The download is 66 MB, and the installation requires 260 MB of free disk space.
If you are not 100% sure of what you're doing, then don't change the defaults. Don't change the default install directory (c:/cygwin), and don't change the default text type (UNIX).

2. Testing

  • Shift-click here to download a test.
  • Save it on your desktop as test.ly (make sure it is not called test.ly.TXT).
  • Double-click the file.
This will process the file, and show the result in a PDF viewer.

Then you did not install the LilyPond package correctly. Go back to 1. Installation and double-check that you've followed the procedure correctly. If in doubt, send mail to lilypond-user@gnu.org

3. Uninstalling

  • By brute force: remove C:\CYGWIN. This also removes your Cygwin /home/ directory, so copy important files to somewhere else first.
  • Officially:

    • Start setup.exe,

    • click Next,
    • select Install from Local Directory.
    • Then click next until you reach Select packages to install.
    • Click View.
    • For every package you wish to uninstall,, click on the recycle icon until it says Uninstall.
    • click Next.
