LilyPond on Microsoft Windows

Installing

  • Download setup.exe by shift-clicking here,
  • For NT, 2000 and XP: become administrator,
  • Run setup.exe,
  • Keep clicking "Next" until...
  • You get to the Package View dialogue. Select LilyPond, found in the Publishing section.
  • LilyPond will now be installed, along with supporting programs. The download is 66 MB, and the installation requires 260 MB of free disk space.

If you are not 100% sure what setup.exe asks, do not change the defaults. Do not change the default install directory (c:/cygwin), and do not change the default text type (type UNIX must be used).

Testing

  • Shift-click here to download a test.
  • Save it on your desktop as test.ly (make sure it is not called test.ly.TXT).
  • Double-click the file.
This will process the file, and show the result in a PDF viewer.

Whoops, something went astray, so you have to send a bugreport. There is a file test.log in the same folder with test.ly. Send this file to lilypond-user@gnu.org with a description of your problem.

Windows 98

On Windows 95, 98 and ME there are problems with the Cygwin Python package, which may hamper the operation of LilyPond (see python fails testsuite). Unfortunately, these problems are outside control of the LilyPond development team. Windows NT, 2000 and XP are not affected.

Uninstalling

  • By brute force: remove C:\CYGWIN. This also removes your Cygwin /home/ directory, so copy important files to somewhere else first.
  • Officially:

    • Start setup.exe,

    • Click Next.
    • Select Install from Local Directory.
    • Click next until you reach Select packages to install.
    • Click View.
    • For every package you wish to uninstall,, click on the recycle icon until it says Uninstall.
    • Click Next.
