LilyPond on MS Windows

1. Installation

  • Download setup.exe by clicking here,
  • For NT, 2000 and XP: become administrator
  • Run setup.exe.
  • Keep clicking "Next" until
  • in the Package View dialogue, select LilyPond, found in the Publishing section.

If you are not 100% exactly sure of what you're doing, then just click "Next". Don't change the default install directory (c:/cygwin), and don't change the default text type (UNIX).

2. Testing

Shift-click href="test.ly">here to download a test. Save it on your desktop, and make sure it's called test.ly (not .TXT). Double-click the file. This will process the file, and show it in a PDF viewer.

Darn, it doesn't work

Whoops, something went astray, so you have to send a bugreport. Run the cywgin terminal (double click the Cygwin icon), and run 

   sh -x /usr/bin/bug-lilypond-cygwin >& bug.txt
Attach the resulting bug.txt file to your bug-report, and send it to bug-lilypond@gnu.org.

3. Uninstalling

  • By brute force: remove C:\CYGWIN. This also removes your Cygwin /home/ directory, so copy important files to somewhere else first.,
  • Officially:

    Start setup.exe, click Next, select Install from Local Directory. Then click next until you reach Select packages to install. Click View. Now, for every package, click on the recycle icon until it says Uninstall. Then, click Next.
