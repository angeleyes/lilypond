LilyPond is included in Debian, so running
apt-get update apt-get install lilypond lilypond-doc
should install LilyPond for you. If you are re running testing or unstable, you should get the latest and greatest. If not, you can file a wish list bug (New upstream release). If you are running woody (stable), see below.
