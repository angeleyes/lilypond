LilyPond Snippet Repository

The Lilypond Snippet Repository (LSR) is a searchable database of Lilypond code snippets. Users may view, search, and contribute their own small snippets of LilyPond code.

Local documentation

The whole documentation can be downloaded as a bzipped tarball at the download page.

If you have installed LilyPond from your GNU/Linux or Unix distribution, documentation is installed in 

  /usr/share/doc/lilypond
You may have to install a separate package such as lilypond-doc.
