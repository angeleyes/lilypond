Documentation is installed in 
  /usr/share/doc/lilypond
You may have to install a separate package such as lilypond-doc.

Support

  • Send comments on these pages to bug-lilypond@gnu.org (mailing list info, search archive)
  • Learn about the latest releases by subscribing to info-lilypond@gnu.org (mailing list info, search archive)
  • Send questions and comments on LilyPond to lilypond-user@gnu.org (mailing list info, search archive)
  • Send bug reports to bug-lilypond@gnu.org (mailing list info, search archive)
  • Contact the developers by subscribing to lilypond-devel@gnu.org (mailing list info, search archive)
  • For contacting the authors, see the author list. (Only contact the authors directly for strictly personal communication)
