February 4, 2004
LilyPond 2.1.18 is out. This release has the new part-combiner installed by default, and a similar implementation of autochange. More information in the release notes. Download here.
February 2, 2004
LilyPond 2.1.17 is out. It adds texts (solo, a due) for the part combiner. It also reinstates the --safe option which prevents havoc by Scheme exploits. More information in the release notes. Download here.
January 28, 2004
LilyPond 2.1.16 is out; its main feature is that it fixes the autobeams gaffe of 2.1.15. The part-combiner has been tested successfully on larger pieces. In the near future, expect more part-combining eye-candy. More information in the release notes. Download here.
January 26, 2004
LilyPond 2.1.15 further improves the part-combiner, and fixes many bugs, among others in pedal brackets and finger positioning. More information in the release notes. Download here.
January 21, 2004
LilyPond 2.1.14 has the first release of the new part combiner. If you have scores that use part-combining, please consider giving it a test-run. In addition many bugs relating to mixed staff sizes have been fixed. More information in the release notes. Download here.
January 20, 2004
The lilypond.org domain has been moved to a new server. This will result in better connectivity and more bandwidth. Due to security concerns, the new server does not offer FTP access, but only HTTP downloads.
January 20, 2004
LilyPond 2.1.13 fixes a small but nasty bug in side-positioning placement, and some bugs in tuplet, tie and accidental formatting. This release contains rudimentary work on a new part-combiner. Download here.

LilyPond 2.1.12 fixes many bugs and improves formatting of ottava brackets. More information in the release notes. Download here.

January 17, 2004
In 2.1.11, the mechanism for setting staff size and page is much simplified. In addition there are improvements in the notehead shape, and there is balloon help! (tadaa!). More information in the release notes. Download here.
January 16, 2004
LilyPond 2.1.10 has a load of bugfixes and a shorter syntax for octave checks. More information in the release notes. Download here.
January 13, 2004
LilyPond 2.1.9 has a new mechanism for adding lyrics to melodies. It is now possible to have different melismatic variations for each stanza. More information in the release notes. Download here.
January 9, 2004
LilyPond 2.1.8 has an important new feature: it is now possible to use \property to tune the appearance of spanning objects like StaffSymbol and SystemStartBrace. In addition, contexts may be modified with \remove and \consists for individual music expressions. More information in the release notes. Download here.

An update to the stable branch, version 2.0.2, was released today. It contains a couple of minor bugfixes. Download here.
