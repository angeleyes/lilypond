LilyPond Fragment Bewaarplaats

De Lilypond Fragment Bewaarplaats (LilyPond Snippet Repository, LSR) is een doorzoekbaar gegevensbestand met LilyPond code fragmenten. Gebruikers kunnen hier hun kleine LilyPond code fragmenten met elkaar delen, bekijken en doorzoeken.

Lokale documentatie

De volledige documentatie kan gedownloaded worden als een ge-bzipped tarbal vanaf de download pagina.

Als je LilyPond hebt geïnstalleerd uit je GNU/Linux of Unix distributie dan is de documentation geïnstalleerd in 

    /usr/share/doc/lilypond
Het zou kunnen zijn dat je daarvoor een apart pakket moet installeren, zoals lilypond-doc.
