LilyPond onder Cygwin

Installatie

  • Download setup.exe door Shift in te drukken en tegelijkertijd hier te klikken,
  • Voor Windows NT, 2000 en XP is het noodzakelijk om vanuit een beheerdersaccount te werken,
  • Start setup.exe,
  • Klik steeds op "Volgende" totdat...
  • de window Package View verschijnt. Selecteer LilyPond, te vinden in de Publishing sectie.
  • LilyPond wordt nu geïnstalleerd samen met de benodigde ondersteuningsprogramma's. Het downloadpakket is 66 MB en voor de installatie is 260 MB vrije schijfruimte nodig.

Als je niet 100 % zeker bent van wat een vraag in setup.exe betekent, verander de standaard antwoorden dan niet. Verander de standaard installatiemap (c:/cygwin) niet, en verander ook het standaard lettertype niet (het UNIX-type moet worden gebruikt).

Testen

  • Druk op Shift en klik tegelijkertijd hier om een test te downloaden.
  • Sla het op je bureaublad op als test.ly (zorg ervoor dat het niet wordt opgeslagen als test.ly.TXT).
  • Dubbelklik op het bestand.
Hierdoor wordt het bestand verwerkt en wordt het resultaat weergegeven in een PDF-viewer.

Verdraaid, het werkt niet

Probeer setup.exe opnieuw op te starten; hierdoor wordt het probleem in 90 % van de gevallen opgelost. Zorg ervoor dat de juiste versie van LilyPond is geïnstalleerd; als je dubbelklikt op het geselecteerde pakket, krijg je een oudere versie.

Als het nog steeds niet werkt, wordt je verzocht een bugreport opsturen. Er is een bestand test.log te vinden in dezelfde map met test.ly. Stuur dit bestand naar lilypond-user@gnu.org met een beschrijving van je probleem.

Windows 98

Op Windows 95, 98 en ME zijn er problemen met het Cygwin Python pakket, die de werking van Lilypond kan belemmeren (zie python fails testsuite). Helaas vallen deze problemen buiten het bereik van het ontwikkelingsteam van Lilypond. In Windows NT, 2000 en XP doen deze problemen zich niet voor.

Ik heb geen test.log

Het LilyPond-pakket is niet correct geïnstalleerd. Ga terug naar Installatie en controleer nogmaals of je de procedure op de juiste wijze hebt uitgevoerd. Als je twijfelt, stuur dan een e-mail naar lilypond-user@gnu.org

Het programma verwijderen

  • De brute methode: verwijder C:\CYGWIN. Hierdoor wordt ook de Cygwin /home/ directory, verwijderd dus kopieer belangrijke bestanden eerst naar een andere plaats.
  • De officiële methode:

    • Start setup.exe,

    • Klik op Volgende.
    • Selecteer Install from Local Directory.
    • Klik op Volgende totdat je Select packages to install bereikt.
    • Klik Bekijken.
    • Voor elk pakket dat je wilt verwijderen geldt: klik op het prullenbak-icoontje totdat de boodschap Verwijderen verschijnt.
    • Klik op Volgende.
