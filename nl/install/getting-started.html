Aan de slag

Je hebt de allernieuwste versie van LilyPond net geïnstalleerd en je kunt niet wachten om aan de slag te gaan met het zetten van muziek. Hier volgen een paar aanwijzingen.

Klik om te vergroten

LilyPond testen

Hier volgen de instructies voor Unix en Windows als je via de commandoregel werkt. Windows-gebruikers wordt geadviseerd om de quickstart-methode voor Windows te raadplegen.

  • Open een terminal window en typ 
    	lilypond --help
    in het scherm. Op het scherm zou het volgende moeten verschijnen 
    	Usage: lilypond [OPTION]... FILE...

	Typeset music and/or produce MIDI from FILE.

	LilyPond produces beautiful music notation.
	For more information, see http://lilypond.org

	Options:
	-b, --backend=EXT     select backend to use
	-e, --evaluate=EXPR   set options, use -e '(ly:option-usage)' for help
	-f, --formats=EXTs    list of formats to dump
	-h, --help            print this help
	[..]

    Is dat het geval, dan is het programma geïnstalleerd en kun je het opstarten.

  • Start je favoriete teksteditor op.
  • Enter 
    	{
	c4( c)
	}
    OPMERKING: gebruikers van LilyPond versie 2.2 of ouder moeten de bovenstaande voorbeeldtekst intypen in \notes{} and \score {} blocks.
  • Opslaan als test.ly. Verlaat de teksteditor.
  • Run 
    	lilypond test
    Als alles goed is, moet de volgende tekst verschijnen 
    	DVI output `test.dvi'...
	PS output to `test.ps'...
	PDF output to `test.pdf'...
  • Bekijk de PDF-file. Probeer de volgende commando's 
    	ggv test.pdf
	gv test.pdf
	kgv test.pdf
	acroread test.pdf
	xpdf test.pdf
	ghostview test.pdf
  • De gemakkelijkste oplossing voor Windows-gebruikers is de drag & drop approach. Als je liever met het toetsenboard werkt, kun je de bovenstaande procedure proberen, met behulp van de volgende punten:
    • Open een shell door dubbel te klikken op het "Cygwin" icoon
    • Start een teksteditor door notepad op te starten. Zorg ervoor dat je bestanden niet worden opgeslagen als test.ly.txt
    • De PDF-file wordt opgeslagen in C: -> CYGWIN -> HOME -> jouw-gebruikersnaam.

Jippie, het werkt!

Als jouw downloadpakket niet door het LilyPond team is samengesteld, stuur dan een kopie van het rapport naar degene die het downloadpakket heeft samengesteld (te vinden in the "Packager"-kolom op de download-pagina).
