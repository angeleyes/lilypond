Avec sa bonne aptitude pour la notation classique, vous pouvez vous demander si LilyPond est utilisable pour la musique contemporaine. La réponse est un oui retentissant.

«Voyez donc, aucun bricolage !»

Si vous générez de la musique avec des logiciels, la publier est aussi simple que d'écrire des fichiers ASCII. Pour la musique contemporaine, vous allez apprécier la souplesse de LilyPond pour sa gestion des mesures et des chiffrages de mesure. Voici un exemple de nolets imbriqués. L'espacement est rigoureusement proportionnel à la durée de chaque note. Les barres de mesures ne gênent en rien, et deux notes ne seront alignées que si elles doivent être attaquées en même temps.

cliquez pour obtenir un fichier PDF.

Avec n'importe quel autre programme, obtenir ce résultat nécessite beaucoup de bricolages périlleux. Avec LilyPond, il suffit de saisir les notes, avec leur structure rythmique : 

\times 7/9 {
  \times 4/6 {
     r8 c32[ c c c c c c c] r4
     c32[ c32 c16
  }
  \times 5/4 {
    c16 c c] c32[ c32 c16 c] r8
  }
}

Si le langage de programmation Scheme vous est familier, vous pouvez même composer par algorithmes avec LilyPond !
