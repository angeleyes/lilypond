LilyPond avec Cygwin sous Microsoft Windows

Installation

  • Téléchargez setup.exe en cliquant ici,
  • Pour Windows NT, 2000 and XP : devenez administrateur,
  • Exécutez setup.exe,
  • Cliquez sur "Next" (ou "Suivant") jusqu'à ce que...
  • Vous obteniez la boite de dialogue Package View. Dans la section Publishing, sélectionnez LilyPond.
  • LilyPond va alors s'installer, avec les programmes nécessaires. La taille du téléchargement est de 66 Mo, et l'installation nécessite 260 Mo d'espace disque libre.

Si vous n'êtes pas totalement sûr de ce que setup.exe vous demande, ne changez pas les paramètres par défaut. Ne changez pas le répertoire d'installation (c:/cygwin), et ne changez pas le codage de caractères (le type UNIX doit être sélectionné).

Test

  • Cliquez ici pour télécharger un test.
  • Enregistrez le sur le bureau, en le nommant test.ly (assurez vous de ne pas le nommer test.ly.TXT).
  • Double-cliquez sur le fichier.
Le fichier sera alors compilé, et le résultat affiché dans un visualisateur de fichiers PDF.

Essayez de relancer setup.exe ; cela résoud 90 % des problèmes qui nous ont été rapportés... Assurez-vous d'avoir installé la bonne version de LilyPond  en cliquant deux fois sur le "package" selectionné, vous obtiendrez une version ancienne de LilyPond.

Si LilyPond ne fonctionne toujours pas, merci d'envoyer un rapport de bug. Un fichier test.log aura été créé dans le même dossier que test.ly. Envoyez ce fichier à lilypond-user@gnu.org avec la description de votre problème.

Windows 98

Le paquet Cygwin Python pose des problèmes sur Windows 95, 98 et ME, risquant de gêner certaines operations de LilyPond (voir les tests python). LilyPond devrait être utilisable, mais la sortie au format PDF en cliquant sur un fichier .ly, lilypond-book, et midi2ly pourraient ne pas fonctionner. Malheureusement, ces problèmes ne sont pas du ressort de l'équipe de développement de LilyPond. Windows NT, 2000 et XP ne sont pas concernés.

Désinstallation

  • La méthode forte : supprimez C:\CYGWIN.

    Le répertoire /home/ de Cygwin sera également supprimé. Faites une copie de sauvegarde des fichiers importants que vous souhaitez conserver.

  • La méthode officielle :

    • Démarrez setup.exe,

    • Cliquez sur "next".
    • Sélectionnez Install from Local Directory.
    • Cliquez sur "next" jusqu'à atteindre Select packages to install.
    • Cliquez sur View.
    • Pour chaque paquet que vous souhaitez désinstaller, Cliquez sur la double fl&eagrave;che jusqu'à l'affichage de Uninstall.
    • Cliquez sur "Next".
