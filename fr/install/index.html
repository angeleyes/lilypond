Notes

Si votre plate-forme n'est pas listée vous pouvez nous aider en réalisant des paquetages LilyPond.

La manière la plus simple d'obtenir Ghostscript 8.x sous Fedora est d'exécuter en tant que super-utilisateur : 
  sudo yum install rpm-build
  rpmbuild --rebuild ftp://dl.xs4all.nl/pub/mirror/fedora/development/SRPMS/ghostscript-8.15.1-1.src.rpm
  rpm -U /usr/src/redhat/RPMS/i386/ghostscript*8.15.1*
