LilyPond avec Fink sous MacOS X

LilyPond est utilisable à travers Fink, dans sa version cvs instable (instable, stable).

L'installation de LilyPond sur MacOS nécessite celle de Fink. Tout est expliqué dans le tutoriel pas-à-pas (plus de détails).

Après cela, vous pourrez tester votre installation.
