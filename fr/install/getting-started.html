Tester LilyPond

Voici les instructions en ligne de commandes, sous Windows et Unix. Les utilisateurs de Windows peuvent jeter un oeil au guide de démarrage pour Windows.

  • Ouvrez un terminal et tapez 
    	lilypond --help
    dans la fenêtre. Vous devriez voir quelque chose comme 
    	Usage: lilypond [OPTION]... FILE...

	Typeset music and/or produce MIDI from FILE.

	LilyPond produces beautiful music notation.
	For more information, see http://lilypond.org

	Options:
	-b, --backend=EXT     select backend to use
	-e, --evaluate=EXPR   set options, use -e '(ly:option-usage)' for help
	-f, --formats=EXTs    list of formats to dump
	-h, --help            print this help
	[..]

    Si vous voyez ça, le programme et installé, vous pouvez l'utiliser.

  • Démarrez votre éditeur de texte favori.
  • Entrez 
    	{
	c4( c)
	}
    NOTE : Avec LilyPond 2.2 et précédents, encadrez l'exemple ci-dessus avec les blocs \notes{} et \score{}.
  • Enregistrez le fichier sous test.ly. Quittez l'éditeur.
  • Exécutez 
    	lilypond test
    Si tout va bien, vous devriez voir 
    	DVI output `test.dvi'...
	PS output to `test.ps'...
	PDF output to `test.pdf'...
  • Ouvrez le fichier PDF pour voir le résultat. Essayez ces commandes : 
    	ggv test.pdf
	gv test.pdf
	kgv test.pdf
	acroread test.pdf
	xpdf test.pdf
	ghostview test.pdf
  • Les utilisateurs de Windows peuvent se servir du glisser / déposer. Si vous préférez le clavier, essayez la procédure ci-dessus, en y ajoutant ces notes :
    • double-cliquez sur l'icone LilyPond ;
    • celà ouvrira un fichier test. Enregistrez le sous test.ly sur le bureau ;
    • double-cliquez sur test.ly ;
    • après quelques secondes, un fichier test.pdf apparaîtra.
