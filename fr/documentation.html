LilyPond Snippet Repository

Le Lilypond Snippet Repository (LSR) est une base de données de recherche d'extraits de code pour LilyPond. Les utilisateurs peuvent consulter, rechercher et contribuer en ajoutant leurs propres extraits de code.

Documentation locale

La documentation est installée dans le dossier 
  /usr/share/doc/lilypond
Il se peut que vous ayez à installer un paquet additionnel tel que lilypond-doc.
