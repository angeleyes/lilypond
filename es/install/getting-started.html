Probando LilyPond

Estas son las instrucciones para Unix y Windows en la línea de comando. Es recomendable para los usuarios de Windows que lean la guía rápida para Windows.

  • Abra una ventana de terminal y escriba 
    	lilypond --help
    en su ventana. Deberá aparecer algo parecido a 
    	Usage: lilypond [OPTION]... FILE...

	Typeset music and/or produce MIDI from FILE.

	LilyPond produces beautiful music notation.
	For more information, see http://lilypond.org

	Options:
	-b, --backend=EXT     select backend to use
	-e, --evaluate=EXPR   set options, use -e '(ly:option-usage)' for help
	-f, --formats=EXTs    list of formats to dump
	-h, --help            print this help
	[..]

    Si aparece esto, tiene el programa instalado y podrá ejecutarlo.

  • Inicie su editor de texto favorito.
  • Inserte 
    	{
	c4( c)
	}
    NOTA: los usuarios de LilyPond 2.2 y anterior deberán encerrar el ejemplo anterior dentro de un bloque \notes{} y a su vez dentro de un bloque \score {}.
  • Guárdelo como test.ly. Salga del editor.
  • Ejecute 
    	lilypond test
    Si ha tenido éxito, deberá aparecer 
    	DVI output `test.dvi'...
	PS output to `test.ps'...
	PDF output to `test.pdf'...
  • Visualice el archivo PDF. Pruebe estos comandos 
    	ggv test.pdf
	gv test.pdf
	kgv test.pdf
	acroread test.pdf
	xpdf test.pdf
	ghostview test.pdf
  • La opción más sencilla para usuarios de Windows es el modo de arrastrar & soltar. Si prefiere el teclado, puede intentar con el procedimiento anterior, con las notas adicionales:
    • doble clic en el ícono de LilyPond
    • esta acción abrirá un archivo de prueba. Guárdelo como test.ly en el escritorio.
    • haga doble clic en test.ly
    • después de algunos segundos, un archivo test.pdf deberá aparecer.
