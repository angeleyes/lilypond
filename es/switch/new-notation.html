Con el objetivo puesto en la notación clásica, usted se preguntará si LilyPond es adecuado para la música moderna. Afortunadamente, la respuesta es un sonoro sí.

“¡Mira, mamá, sin trucos!”

Si usted produce música mediante software, publicarla es tan fácil como escribir unos sencillos archivos de texto ASCII. Si usted hace música de vanguardia, entonces adorará la flexibilidad de Lily con las medidas y las indicaciones de compás. He aquí una muestra con grupos anidados. El espaciado es exactamente proporcional ala duración de cada nota. Las notas no han de llenar los compases, y dos notas solamente se alinean si comienzan juntas exactamente.

Pulse con el ratón para descargar un archivo PDF.

En cualquier otro programa, conseguir este resultado necesita una desmesurada cantidad de trucos y artimañas especiales. En LilyPond, sólo hay que introducir las notas, con su estructura rítmica: 

\times 7/9 {
  \times 4/6 {
     r8 c32[ c c c c c c c] r4
     c32[ c32 c16
  }
  \times 5/4 {
    c16 c c] c32[ c32 c16 c] r8
  }
}

Si usted está familiarizado con el lenguaje de programación Scheme ¡podrá incluso componer de forma algorítmica desde dentro de LilyPond!

“¡Mira, mamá, qué trucos tan fáciles!”
