Captura de pantalla

LilyPond es un sistema de lotes: se ejecuta sobre un fichero de texto que contiene las notas. La salida resultante se podrá visualizar en la pantalla o imprimirse. La captura de pantalla de un caso de utilización real podría tener este aspecto:

Es tan fácil como "A, B, C"

Las notas se codifican mediante letras y números. Las instrucciones especiales se escriben mediante barras invertidas «\».


Las alteraciones se hacen con nombres diferentes: añada -is para el sostenido, y -es para el bemol (estos son los nombres de las notas en el idioma danés, pero puede usar otros idiomas). LilyPond sabrá cómo colocar las alteraciones accidentales.
