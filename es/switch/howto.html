LilyPond es un sistema de lotes: se corre sobre ficheros de texto que contienen las notas. La salida resultante se podrá visualizar en pantalla o imprimir. Una captura de pantalla del uso práctico podría verse así

Es tan fácil como A B C

Las notas se escriben con letras y números. Los comandos especiales pueden insertarse con diagonales invertidas "\".


Las alteraciones se hacen con diferentes nombres: añada -is para sostenido, y -es para bemol (estos son los nombres Daneses, pero puede usar otros idiomas). LilyPond sabrá cómo colocar los accidentes.
