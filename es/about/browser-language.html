Usando selección autmática de idioma

Para obtener la selección autmática de idioma, deberá indicarle a su navegador, cuál es su idioma preferido. La forma de hacerlo dependerá del navegador que esté utilizando.

Mozilla Firefox versión 0.9 y más recientes
GNU/Linux 
      Editar -> Preferencias -> Avanzadas -> Idiomas
Microsoft Windows 
      Herramientas -> Opciones -> General -> Idiomas
In older version you have to go to about:config and change the value of intl.accept_languages.
Mozilla / Netscape 4.x and newer
      Edit -> Preferences -> Navigator -> Languages
Nota: con Netscape 4.x necesitará asegurarse de seleccionar el idioma a partir de las alternativas posibles. Algunas personas han reportado problemas por que escribieron el idioma a mano.
Microsoft Internet Explorer
Microsoft Windows 
      Herramientas (o Vista o Extras) -> Opciones de Internet -> (General) Idiomas
      Tools (or View or Extras) -> Internet Options -> (General) Languages
MacOS 
      Editar -> Preferencias -> Navegador Web -> Idioma/Fuentes
