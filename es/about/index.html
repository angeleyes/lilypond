LilyPond es impresión de música

LilyPond es un sistema automatizado de grabado de música. Da formato de forma bella y autmática, y tiene una sintaxis amigable para sus archivos de entrada. Es software Libre (‘open source’).

LilyPond es una parte del Proyecto GNU.


Los desarrolladores cooperan con el grupo de Desarrolladores de audio de Linux, y forman parte del Consorcio de Audio Linux.

LilyPond utiliza el intérprete SCHEME del Proyecto GNU GUILE para extensiones. Scheme es un dialecto del lenguaje de programación LISP.
