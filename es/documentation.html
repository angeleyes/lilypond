Repositorio de fragmentos de código de LilyPond

El repositorio de fragmentos de código de Lilypond (Lilypond Snippet Repository, LSR) es una base de datos con capacidad de búsqueda que contiene fragmentos de código Lilypond. Los usuarios pueden ver, buscar y aportar sus propios fragmentos de código LilyPond.

Documentación local

Toda la documentación se puede descargar como un archivo tar comprimido con bzip2 a partir de la página de descargas.

Si ha instalado LilyPond a partir de su distribución de GNU/Linux ó Unix, la documentación está instalada en 

  /usr/share/doc/lilypond
Puede ser que deba instalar un paquete separado como lilypond-doc.
