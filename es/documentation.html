Repositorio de fragmentos de código LilyPond

El repositorio de fragmentos Lilypond (Lilypond Snippet Repository LSR) es una base de datos con capacidad de búsqueda que contiene fragmentos de código Lilypond. Los usuarios pueden ver, buscar y contribuir sus propios fragmentos de código LilyPond.

Documentación local

La documentación se instala en 
  /usr/share/doc/lilypond
Puede ser que deba instalar un paquete separado como lilypond-doc.
