LilyPond in Cygwin unter Microsoft Windows

Installation

  • Lade setup.exe herunter indem du, w&aauml;hrend du die Shift-Taste drückst, hier klickst,
  • für NT, 2000 und XP musst du Administratorrechte besitzen.
  • Starte setup.exe,
  • fahre fort "Next" (weiter) zu klicken, bis...
  • du zum Paketansicht-Dialogfenster gelangst. W&aauml;hle LilyPond aus, welches du im Abschnitt Publishing findest.
  • LilyPond wird nun installiert werden, gleichzeitig mit unterstützenden Programmen. Der download ist 66 MB groß, die Installation benötigt 260MB freien Festplattenplatz.

Wenn du dir nicht 100% sicher bist was setup.exe dich fragt, ver&aauml;ndere die Voreinstellungen nicht. Ver&aauml;ndere nicht das voreingestellte Installationverzeichnis(c:/cygwin) und belasse den voreingestellten Texttyp (Typ UNIX muss eingestellt sein).

Test

  • Shift-klicke hier um den Test herunterzuladen.
  • Speichere die Datei auf dem Desktop unter dem Namen test.ly (stelle sicher, dass sie nicht test.ly.TXTgenannt wird.
  • Klicke doppelt auf der Datei.
Das wird die Datei verarbeiten und das Ergebnis in einem PDF-Betracther anzeigen.

Windows 98

Auf Windows95, 98 und ME gibt es Probleme mit dem Cygwin Paket, welches die fehlerfreie Operations von LilyPond stören kann. (siehe Die Python test-suite schl&aauml;gt fehl ). LilyPond selbst sollte benutzbar sein, aber das Klicken auf eine .ly> Datei, dass ein PDF erzeugt. Desweiteren kann es sein, dass lilypond-book und midi2ly nicht funktionieren. Unglücklicherweise sind diese Probleme außerhalb der Reichweite des LilyPond-Entwicklerteams. Windows NT, 2000 und XP sind nicht betroffen.

Deinstallation

  • By brute force (Unter zuhilfenahme roher Gewalt): entferne C:\CYGWIN.

    Das entfern auch dein Cygwin /home/ Verzeichnis, also kopiere wichtige Dateien vorher woandersthin.

  • Offizieller Weg:

    • Starte setup.exe,

    • Klicke auf Next.
    • W&aauml;hle Install from Local Directory. (Installiere von lokalem Verzeichnis)
    • Klicke auf next bis du den Dialog Select packages to install (Paketinstallationsauswahldialog) erreichst.
    • Klicke auf View.
    • Für jedes Paket, dass du deinstallieren willst klicke auf das Recycleicon bis es Uninstall sagt.
    • Klicke auf Next.
