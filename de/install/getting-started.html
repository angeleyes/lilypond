LilyPond testen

Dies sind Anweisungen für Linux und Windows für die Kommandozeile. Windowsbenutzer wird es geraten sich den Schnellstart für Windowsanzusehen.

  • öffne ein Terminalfenster und gib 
    	lilypond --help
    in dein Fenster ein. (Mit der Entertaste abschließen) Du solltest etwas &aauml;hnliches wie 
    	Usage: lilypond [OPTION]... FILE...

	Typeset music and/or produce MIDI from FILE.

	LilyPond produces beautiful music notation.
	For more information, see http://lilypond.org

	Options:
	-b, --backend=EXT     select backend to use
	-e, --evaluate=EXPR   set options, use -e '(ly:option-usage)' for help
	-f, --formats=EXTs    list of formats to dump
	-h, --help            print this help
	[..]
    sehen.

    Wenn du das siehst ist LilyPond installiert und du kannst es benützen.

  • Starte deinen Lieblingseditor (z.B notepad ;)
  • Gib 
    	{
	c4( c)
	}
    ein. Hinweis: Benutzer von LilyPond 2.2 and früherer Versionen müssen das oben angeführte Beispiel in \notes{} und \score {} Blocks einschließen..
  • Speichere ihn unter dem Namentest.ly. Beende den Editor.
  • Gib im Terminal/Konsole 
    	lilypond test
    ein Wenn alles funktioniert, sollte die Ausgabe 
    	DVI output `test.dvi'...
	PS output to `test.ps'...
	PDF output to `test.pdf'...
    enthalten.
  • Lass die PDF-Datei anzeigen. Probiere folgende Kommandos aus: 
    	ggv test.pdf
	gv test.pdf
	kgv test.pdf
	acroread test.pdf
	xpdf test.pdf
	ghostview test.pdf
  • Der einfachste Weg für Windowsbenutzer ist die drag & drop Verfahrensweise (Hier Doppelklick auf der .ly Datei). Wenn du die Tastatur bevorzugst, kannst du auch das obige Verfahren verwenden, mit den folgenden Tips:
    • &Oum;ffne eine Shell indem du auf dem "Cygwin"-Icon doppelt klickst. icon
    • Starte einen Texteditor z.B durch starten von notepad (Start-> Asuführen dann notepad eingeben) . Speichere deine Eingaben unter test.ly und stelle sicher, dass notepad kein .txt/ anh&aauml;ngt (Entwede nenne den schließe den Namen in "" ein oder stelle Den Dateitypauswahlfilter auf "alle Dateien")
    • Die PDFdatei wird in C: -> CYGWIN -> HOME -> your-user-name belassen.
